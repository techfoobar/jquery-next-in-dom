jQuery Next-In-DOM

jQuery Next-In-DOM is a jQuery plugin for getting the next or previous element in the full document matching a specified selector from an element. It basically traverses the whole DOM from the pivot element to find a matching element and returns the same if found. This is different from jQuery's built-in next() method which only returns the next or previous (matching) sibling of an element.
Tested on Firefox, Google Chrome and IE7 (and up). If you have any suggestions, comments or if you have found a bug, please send it to vjeleven@gmail.com

Usage
  1. Include the script on your page:
    <script src="/path/to/nextindom.jquery.min.js"></script>
  2. Once the DOM is ready (or anytime thereafter), you can call the methods:
    
// get the previous element from the element matching 'selector1' (but not including)
// regardless of the DOM structure
var prev1 = $('selector1').prevInDOM();

// get the previous element from 'selector1' matching 'selector2'
var prev2 = $('selector1').prevInDOM('selector2');
    
// get the next element from the element matching 'selector1' (but not including)
// regardless of the DOM structure
var next1 = $('selector1').nextInDOM();

// get the next element from 'selector1' matching 'selector2'
var next21 = $('selector1').nextInDOM('selector2');
